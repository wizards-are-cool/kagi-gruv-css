# kagi-gruv-css
Dark Gruvbox CSS style for Kagi.


## How to Install

1. Open Kagi and go to `Settings > Appearance`.
2. Change theme to `Dark` theme if it isn't already selected.
3. Go to `Custom CSS` section.
4. Toggle `Enable CSS` button.
5. Copy and paste [kagi.css](/kagi.css) into the custom style box. 
6. Click `Save Changes`.

> [!NOTE]
> This style is incomplete and assumes you only have the Wikipedia search widget enabled. Also, I've only tested it on Firefox 146.0.1 and mobile Chromium (GrapheneOS Vanadium). 

## Tweaks
### Change Font
Change font in `--font-main`. 
### Remove AI Features
Append [kagi-no-ai](https://codeberg.org/zerodogg/kagi-no-ai/raw/branch/main/kagi-no-ai.css) to this theme.
